# NMC EMR React Migration Analysis

NMC EMR 시스템의 React 마이그레이션을 위한 분석 프로젝트 저장소

## 프로젝트 목표

**최종 목표**: EMR 시스템 React 마이그레이션을 위한 **분석 완료** (React 전환 이전 단계까지)

- React Migration 프로젝트 접근법 파악
- 전략 수립 및 실행 계획 수립
- 프론트엔드 이슈 사전 점검
- 백엔드 전환 필요성 평가

## 중요 조건 (Go/No-Go 기준)

| 기준 | 임계값 | 초과 시 조치 |
|------|--------|-------------|
| **백엔드 전환율** | 10% | **프로젝트 중단** |
| **React 구현 어려움** | 핵심 기능 2건 이상 | 대안 기술 검토 |

## 저장소 구조

```
nmc-emr-analysis/
├── README.md              # 프로젝트 개요
├── .gitignore             # Git 무시 파일
├── analysis/              # 화면 분석서
│   ├── SMRPE/            # 채용/인사 모듈
│   ├── SMPMO/            # 외래진료 모듈
│   ├── SMPMI/            # 입원/처방 모듈
│   ├── MMO/              # 처방/오더 모듈
│   └── SPR/              # 공통 팝업
├── design/                # React 설계안
├── scripts/               # 자동화 스크립트
├── reports/               # 보고서
└── docs/                  # 문서/가이드
    ├── 작업계획서.md
    ├── 팀운영계획서.md
    └── templates/         # 분석 템플릿
```
