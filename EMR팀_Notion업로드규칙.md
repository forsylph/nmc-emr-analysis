## 🔄 처리 Flow

```
┌─────────┐    ┌─────────┐    ┌─────────┐    ┌─────────┐
│  작업   │ → │  완료   │ → │  자동   │ → │  검토   │
│  시작   │    │  체크   │    │  업로드 │    │  완료   │
└─────────┘    └─────────┘    └─────────┘    └─────────┘
     ↑                                       ↓
     └─────────── 다음 작업 ←──────────────┘
```

**상세 절차**:
1. 팀원이 작업 완료
2. 마크다운 문서 작성 (템플릿 준수)
3. 자동으로 Notion에 업로드 실행
4. 시디/피디가 검토
5. PM이 전체 진행 상황 수집
